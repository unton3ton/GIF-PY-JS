📦 GIF → Кадры (ZIP)

Загрузите GIF, чтобы получить архив со всеми кадрами в формате PNG.

🎞️ Кадры → GIF

Загрузите несколько изображений (PNG, JPG, GIF и др.). Файлы будут отсортированы по цифрам в имени.

📌 Для правильного порядка используйте имена вида frame_1.png, frame_2.png или frame_001.png (числа в любом месте имени).
Задержка между кадрами (Duration) 50 ms
⚠️ Не более 200 файлов, каждый до 20 МБ.
